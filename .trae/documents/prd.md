## 1. 产品概述

精准农业 IoT 全栈监控平台——面向大型农场的实时土壤感知与空间决策系统。系统通过订阅成百上千个 LoRa 土壤探针的高频 MQTT 数据流，经按位解包还原氮磷钾（NPK）、电导率与深层温度，写入 InfluxDB 时序数据库，并在大屏端以 WebGL 热力图叠加 GeoJSON 地块轮廓的方式，实现 5 分钟级降采样数据的流式可视化。

- **目标用户**：农场管理者、农艺师、精准农业决策团队
- **核心价值**：将离散的土壤传感数据转化为直觉可读的空间养分图谱，支撑变量施肥与灌溉决策

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|

## 3. 核心流程

### 3.1 MQTT 数据摄取链路

LoRa 探针 → EMQX Broker → NestJS 订阅 → Hex 解包（NPK/EC/Temp） → 批量写入 InfluxDB

```mermaid
flowchart LR
    "LoRa探针群" -->|"MQTT Publish"| "EMQX Broker"
    "EMQX Broker" -->|"MQTT Subscribe"| "NestJS MQTT Service"
    "NestJS MQTT Service" -->|"Hex解包"| "Payload Parser"
    "Payload Parser" -->|"NPK/EC/Temp + Timestamp"| "InfluxDB Writer"
    "InfluxDB Writer" -->|"批量写入"| "InfluxDB"
```

### 3.2 大屏流式回显链路

InfluxDB → NestJS 聚合查询 → WebSocket 推送 → Vue3 热力图/图表平滑更新

```mermaid
flowchart LR
    "InfluxDB" -->|"5min降采样查询"| "NestJS WS Gateway"
    "NestJS WS Gateway" -->|"WebSocket Push"| "Vue3 前端"
    "Vue3 前端 -->|"WebGL渲染"| "热力图渐变"
    "Vue3 前端" -->|"ECharts更新"| "仪表盘/曲线"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深空墨绿（#0A1F1C）作为背景底色，象征土壤与自然；琥珀金（#D4A843）作为强调色，代表成熟与收获
- **辅助色**：翠绿（#2ECC71）用于正常状态，赤橙（#E74C3C）用于告警状态
- **字体**：数据展示使用 JetBrains Mono（等宽、科技感），标题使用 Noto Sans SC
- **布局**：全屏暗色大屏，左侧地图区域占 60%，右侧数据面板占 40%，顶部状态栏
- **按钮风格**：扁平化，圆角 4px，悬停时发光边框
- **图标风格**：线性描边，24px，与数据大屏科技风格统一

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 监控大屏 | 地块热力图 | 深色地图底图 + 绿-黄-红色阶热力层 + 地块描边轮廓 + 探针标注点 |
| 监控大屏 | NPK 仪表盘 | 三个弧形仪表盘（N/P/K）+ 底部5min趋势折线图 + 阈值虚线 |
| 监控大屏 | 告警状态栏 | 顶部横条，左侧告警图标 + 滚动文字，右侧时间戳 |
| 监控大屏 | 探针在线率 | 右上角环形图 + 中心数字 + 底部图例 |
| 监控大屏 | 指标聚合卡片 | 4列卡片，每卡片含图标+数值+迷你趋势图 |

### 4.3 响应式

- 桌面优先（≥1920px 全屏大屏最优）
- 1366px–1919px 自适应缩放
- 触摸屏兼容（农场现场大屏触控操作）

### 4.4 动效设计

- 热力图色斑：CSS transition + requestAnimationFrame 双缓冲插值，实现数据更新时 800ms 渐变扩散
- 仪表盘指针：弹性缓动（cubic-bezier）
- 告警条目：入场 slideIn + 3s 自动消失
- 数据卡片数值：数字滚动动画（countUp）
